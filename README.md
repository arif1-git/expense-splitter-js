# 💸 MySpend - Expense Splitter

A simple and intuitive **Expense Splitter Web Application** built using **HTML, CSS, and JavaScript**. The application helps a group of **three friends** easily track shared expenses and calculate who owes whom, making group outings and trips hassle-free.

🌐 **Live Demo:** https://myspend2.netlify.app/

## 🚀 Project Overview

**MySpend** is a lightweight finance utility designed to simplify expense sharing among friends. Users can record expenses, assign who paid, and automatically calculate the settlement amount for each participant.

The project was built to practice JavaScript logic, DOM manipulation, and interactive frontend development by solving a real-world problem.

## ✨ Features

* 👥 Manage expenses for a group of three friends
* ➕ Add shared expenses
* 💰 Record who paid for each expense
* 🧮 Automatic expense and settlement calculation
* 📊 Display final balances and payment summaries
* ✏️ Edit and update expense entries
* 🗑️ Delete expenses when needed
* 📱 Responsive design for desktop and mobile devices
* ⚡ Fast, lightweight, and browser-based

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript
* **Logic:** Vanilla JavaScript
* **Storage:** Browser Local Storage (if enabled)
* **Deployment:** Netlify
